## 🤔 O que é o Cloakfy?

O Cloakfy é uma plataforma completa de **cloaking e camuflagem de criativos** para profissionais de tráfego pago. Ele protege suas páginas de vendas mostrando conteúdos diferentes para visitantes diferentes:

- **Bots e moderadores** → veem uma página segura (White Page)
- **Leads reais** → veem sua oferta verdadeira (Black Page)

Além disso, o Cloakfy camufla seus criativos (vídeos, imagens e áudios) alterando metadados e propriedades dos arquivos para evitar rejeição nas plataformas de ads.

## ✨ Funcionalidades Completas

### 🔮 Cloaker de Páginas
- Wizard intuitivo de 3 etapas (Campanha → Páginas → Filtros)
- Suporte a **Meta Ads, TikTok Ads, Google Ads, Kwai, Taboola, MGID**
- Detecção automática de bots por **User Agent, ASN e Fingerprint**
- Listas de bots específicas por plataforma (Facebook, Google, TikTok, Kwai)
- Redirect 302 transparente (funciona com qualquer site)
- Token XGO — chave de segurança na URL que protege contra acesso direto
- Preview Bot vs Humano — veja lado a lado o que cada um vê

### 📹 Camuflagem de Criativos
- **Camuflar Vídeo** — altera metadados, hash, codec de vídeos
- **Camuflar Imagem** — altera EXIF, GPS, dados de câmera
- **Camuflar Áudio** — altera metadados de arquivos de áudio
- **🎵 Camuflagem TikTok** — otimizada pro TikTok com 3 níveis:
  - ⚡ Básica: metadados + hash
  - 🔧 Normal: + framerate + bitrate variado
  - 🔥 Agressiva: + micro-crop + re-encode completo
- Processamento 100% no navegador (seus arquivos nunca saem do seu PC)

### 🛡️ Proteção Anti-Bot
- Detecção por User Agent (Facebook, Google, TikTok, spy tools)
- Detecção por ASN (redes do Meta, Google)
- JS Challenge (Canvas/WebGL fingerprinting)
- Bloqueio de VPN/Proxy
- Bloqueio de spy tools (AdSpy, BigSpy, SimilarWeb, SEMrush)
- Token XGO — chave secreta obrigatória na URL
- Cloaker Automático — IA decide quem vê o quê

## ⚡ Tecnologia: Smart SSR Routing

O Cloakfy utiliza **Renderização Dinâmica por User-Agent** — uma tecnologia de roteamento inteligente no edge que serve conteúdo diferente dependendo de quem acessa:

```
Request → Vercel Edge → Detecta User-Agent
                          │
                          ├─ 🤖 Bot (Googlebot, Facebook, TikTok)
                          │    → HTML completo renderizado no servidor
                          │    → Meta tags, JSON-LD, Open Graph
                          │    → Página Safe (White Page)
                          │
                          └─ 👤 Humano (Chrome, Safari, Firefox)
                               → SPA React (Single Page App)
                               → Página Real (Black Page)
```

Essa arquitetura combina o melhor dos dois mundos:
- **SEO perfeito** — bots de busca indexam conteúdo completo (blog, landing page)
- **Performance máxima** — humanos recebem SPA rápido com React
- **Cloaking transparente** — bots de ads veem safe page sem redirect detectável
- **Zero latência extra** — processamento no edge (Vercel Edge Functions), não em servidor distante

Utilizado em dois módulos:
1. **Blog SSR** — Googlebot recebe HTML com artigos completos + JSON-LD Article
2. **Cloaker de Páginas** — bots de plataformas de ads recebem White Page, humanos recebem oferta real
